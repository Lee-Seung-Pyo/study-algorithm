## BOJ4889 - 안정적인 문자열  
(문제: https://www.acmicpc.net/problem/4889)  
(정답: https://www.acmicpc.net/source/19913920)  
(시간: 20분)  

스택 문제이지만 굳이 스택으로 안풀어도 풀 수 있는 문제. 연산을 행하는 최소값을 구하래서 이걸 어떻게 구하나 하고 쫄았는데 그냥 평소 스택으로 괄호 확인하듯이 해주면 된다.  

우선 안정적인 문자열이 안되는 경우는 두가지이다.  
1. '{' 가 많거나 → 끝났을 때 '{' 가 남는 경우 → 남은 개수/2 해주면 됨  
2. '}' 가 많거나 → 중간에 비어있는데 '}' 가 들어오는 경우 → '{' 로 바꿔주고(연산 +1) 계속 검사  

연산의 최소 횟수라 해서 헷갈릴 수 있는데 어차피 2번의 경우는 앞에 놈 중에 '}' 하나를 바꿔주든 지금꺼를 바꿔주든 해야하기 때문에 연산은 무조건 한번 해야되는거고, 그렇게 검사 다 마쳤을 때 남은 '{' 의 절반은 바꿔줘야 하기 때문에 남은 개수/2 해주는거.
