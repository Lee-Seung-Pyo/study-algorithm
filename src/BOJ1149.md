## BOJ1149 - RGB거리  
(문제: https://www.acmicpc.net/problem/1149)  
(정답: https://www.acmicpc.net/source/20133252)  
(시간: 55분)  

DP 점화식 유도하는게 어렵다.ㅠㅠ  

단순히 'dp[n] = n까지 칠할 때의 최솟값' 으로 두면 유추해낼 수가 없다.  
dp[n][0] = n번째를 빨강으로 칠할 때의 최솟값 = min(dp[n-1][1], dp[n-1][2]) + n번째를 빨강으로 칠하는 비용  
dp[n][1] = n번째를 초록으로 칠할 때의 최솟값 = min(dp[n-1][0], dp[n-1][2]) + n번째를 초록으로 칠하는 비용  
dp[n][2] = n번째를 파랑으로 칠할 때의 최솟값 = min(dp[n-1][0], dp[n-1][1]) + n번째를 파랑으로 칠하는 비용  

n까지 칠할 때의 최솟값은 min(dp[n][0], dp[n][1], dp[n][2]) 이다.
