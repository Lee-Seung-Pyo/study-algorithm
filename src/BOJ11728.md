## BOJ11728 - 배열 합치기  
(문제: https://www.acmicpc.net/problem/11728)  
(정답:  
stl sort - https://www.acmicpc.net/source/20076016  
Merge sort - https://www.acmicpc.net/source/20076064)  
(시간: 6분)  

Merge Sort 연습문제.

stl \<algorithm>의 sort를 이용하면 최악의 경우 O(NlogN)이다.  
Merge Sort의 합치는 과정을 통하면 O(N)이다.  
제출해보니 전자는 668ms, 후자는 524ms가 나옴. 생각보다 차이가 크진 않다.
